# 크레스티드 게코 패턴·색상·구조 형질 — 종합 가이드

> 조사일: 2026-04-14  
> 관련 종: 크레스티드 게코 (Correlophus ciliatus)  
> 성격: 대부분 **다유전자(polygenic)/선별교배** 형질. 단일 유전자 모프와 달리 멘델 유전을 따르지 않으며, 세대에 걸친 선별 교배로 형질을 강화.  
> 예외: **팬텀(Phantom)** — 최근 열성(recessive) 단일 유전자로 증명됨

---

## A. 패턴 형질 (Pattern Traits) — 7그룹 분류

> 많은 패턴 형질이 "별개 모프"처럼 이름이 붙어 있지만, 실제로는 같은 축의 **연속 스펙트럼**이거나 **파생 변형**인 경우가 대부분이다. 아래는 실질적 독립 그룹 기준으로 재분류한 것.

---

### 그룹 1. 플레임–할리퀸 계열 (패턴 커버리지 스펙트럼)

밝은 패턴이 몸을 **얼마나 넓게 덮는가**에 따라 같은 축 위의 등급이 달라진다. 유전은 모두 다유전자/선별교배.

```
패턴리스/바이컬러 ← 플레임 ← 할리퀸 ← 익스트림 할리퀸
   (0%)            (등면만)   (등+측면+다리)  (75%+ 커버리지)
```

| 등급 | 정의 | 외형 | 가격 (USD) |
|------|------|------|-----------|
| **패턴리스 (Patternless)** | 전신 단색, 패턴 없음. 최고 품질은 색상 변동 0% | 단색 | $50~$200 |
| **바이컬러 (Bicolor)** | 패턴리스와 유사하나 등면이 나머지와 **약간 다른 톤** | 2톤 | $50~$200 |
| **플레임 (Flame)** | 등면에 밝은 스트라이프 + 측면·다리에 패턴 거의 없음 | 등면만 밝음, 대비 뚜렷 | $50~$200 |
| **할리퀸 (Harlequin)** | 플레임에서 **측면·하부·다리까지 패턴 확장** | 등면 + 측면에 크림/흰색 고르게 분포 | $150~$400 |
| **익스트림 할리퀸 (Extreme Harlequin)** | 측면 패턴이 상부까지 올라와 등면과 합류 (**75%+** 커버리지) | 밝은 색이 몸 대부분 차지 | $400~$700 |

**트라이컬러 (Tricolor)** — 이 계열의 **색상 변형**. 할리퀸/익스트림 할리퀸에서 **3가지 뚜렷이 구분되는 색상**이 균등 분포.

| 항목 | 내용 |
|------|------|
| **대표 조합** | (1) 레드/오렌지 + 크림/화이트 + 다크 브라운/블랙, (2) 라벤더 + 옐로우/오렌지 + 화이트 |
| **유전** | 다유전자/선별교배. 다세대 계획·신중한 선별 필요 |
| **가격** | $350~$700. 대칭적 패턴 분포 시 프리미엄 |
| **시장** | 2026년 현재 전용 브리딩 커뮤니티가 활발 |

**브리딩 참고**: 할리퀸 × 할리퀸 → 자손 중 일부 익스트림, 일부 플레임, 일부 바이컬러 (변동 있음). 최고 패턴 개체끼리 교배하면 확률은 높아지지만 보장되지 않음.

### 그룹 2. 타이거–브린들 계열 (줄무늬 연속성 스펙트럼)

등면의 **수직 줄무늬가 얼마나 연속적인가**에 따라 나뉜다. 같은 형질의 표현 강도 차이.

```
브린들 (파편화) ← → 타이거 (연속 줄)
```

| 등급 | 정의 | 외형 | 가격 (USD) |
|------|------|------|-----------|
| **타이거 (Tiger)** | 등면에서 측면으로 내려가는 **연속적 수직 줄무늬** | 적갈색/오렌지/황색 호랑이 무늬. 양쪽에서 등면을 가로지름 | $100~$300 |
| **브린들 (Brindle)** | "불규칙하게 깨진 타이거" — 줄무늬가 끊기고 점·마블·격자로 파편화 | 레이스워크/격자 느낌 | $100~$300 |

**구분 핵심**: 타이거 = 선형(linear), 브린들 = 파편화(fragmented). 혼동이 잦지만 본질적으로 같은 형질의 연속 스펙트럼.

---

### 그룹 3. 핀스트라이프 계열 (돌출 스케일 줄)

등 양옆의 **돌출(raised) 스케일 줄**을 기반으로 한 형질. 줄의 수·두께·배치에 따라 변형이 갈린다. 독립 형질이라 플레임, 할리퀸, 타이거 등 **어떤 패턴과도 결합 가능**.

| 변형 | 정의 |
|------|------|
| **핀스트라이프 (Pinstripe)** | 등 양옆 크레스트 끝 → 꼬리 기저까지 밝은 돌출 스케일 **2줄**. 풀 핀(완전) vs 파셜 핀(끊김) |
| **쿼드스트라이프 (Quadstripe)** | 핀스트라이프 2줄 + 등 중앙 추가 2줄 = **총 4줄** |
| **슈퍼스트라이프 (Superstripe)** | 핀이 극도로 두꺼워지고 등면 중앙에 어두운 줄이 남는 형태 |
| **엠프티 백 (Empty Back)** | 핀스트라이프 줄은 밝지만 그 사이 등면 전체가 기저색으로 어둡게 채워짐 → "빈 등" |
| **리버스 핀 (Reverse Pin)** | 등면 패턴 아래쪽에 어두운 줄 |
| **인버티드 핀 (Inverted Pin)** | 등면 패턴 안쪽에 어두운 줄 |
| **핀대시 (Pin-dashed)** | 핀스트라이프가 점선 형태로 끊김 |

| 항목 | 내용 |
|------|------|
| **유전** | 독립 형질 (다유전자) |
| **가격** | $400~$700 (하이 커버리지 핀스트라이프 기준) |

### 그룹 4. 스팟 계열 (점 패턴)

몸에 분포하는 **점(spot)**의 유형에 따라 나뉜다.

| 형질 | 정의 | 비고 |
|------|------|------|
| **달마시안 (Dalmatian)** | 검은 점 25개 이상. **슈퍼 달마시안** = 100개+. 잉크 스팟(대형), 컨페티(빨간/흰 점) 변형 존재 | 해칭 시 점 적음 → 25g에서 본격 발달 |
| **스노우플레이크 (Snowflake)** | **흰색** 반점이 눈송이처럼 흩뿌려진 형태 | 달마시안과 반대로 밝은 점 |

| 항목 | 내용 |
|------|------|
| **유전** | 불명확. 달마시안 × 달마시안 = 달마시안 자손 경향이 있으나 단일 유전자 미증명 |
| **가격** | $120~$250 (스탠다드 달마시안), 슈퍼 달마시안은 더 높음 |

## B. 팬텀 (Phantom) — 증명된 열성 유전 모프

> ⚡ 이 형질만 **단일 유전자, 열성(recessive)**으로 증명됨. 아잔틱에 이어 크레스티드 게코의 2번째 증명된 열성 형질.

### 역사

- 오랫동안 브리더들이 "이상한 패턴리스/바이컬러"로 분류하며 혼동
- 레드 × 레드 교배에서 예기치 않게 나타나는 어두운 자손 → 팬텀 het이 몰래 퍼져 있었기 때문
- 최근 수년간의 체계적 번식 프로젝트로 **열성 유전 확정**

### 유전학

| 교배 조합 | 예상 결과 |
|-----------|-----------|
| Phantom × Phantom | 100% Phantom |
| Phantom × Het Phantom | 50% Phantom, 50% Het |
| Het × Het | 25% Phantom, 50% Het, 25% Normal |
| Phantom × Normal | 100% Het (외형 노멀) |

- Het 팬텀은 **외형으로 판별 불가** → 계통 추적 필수

### 외형

| 항목 | 특징 |
|------|------|
| **기저색** | 멜라닌 증가 → **어둡고 뮤트된** 기조. 회색, 갈색, 거의 검정 |
| **패턴** | 흰색/크림 패턴이 **억제·탈색** (washed out) |
| **파이어링** | 파이어드/언파이어드 모두 **동일하게** 어둡고 억제된 상태 유지 |
| **꼬리 기저** | 희미한 흰색 패치 (꼬리 기저·배 근처) |

### 주요 콤보

| 콤보 | 특징 | 가격 (USD, 2025) |
|------|------|-----------------|
| **팬텀 핀스트라이프** | 뮤트된 등면 + 흰색 핀스트라이프 유지 → 미묘한 대비 | $200~$400 |
| **팬텀 릴리 (Phantom Lilly)** | 고대비: 팬텀의 어둠 + 릴리의 흰색 | $400~$900 |
| **팬텀 아잔틱** | 거의 모노크롬. "유령 같은" 외형 | $1,000~$2,000 |
| **팬텀 트라이컬러 (Pink Phantom)** | 뮤트된 3색 조합 | $400~$800 |
| **팬텀 카푸치노** | 이중 어둠 효과 | 프리미엄 |
| **팬텀 프라푸치노** | 카푸치노 + 릴리 + 팬텀 3중 | 매우 희귀 |

### 시장 가치 변천

| 시기 | 가격 | 인식 |
|------|------|------|
| 2020년대 초반 | $150~$225 | "별로 안 예쁜 패턴리스" 취급 |
| 2023~2024 | $200~$400 | 열성 증명 후 급등, 콤보 수요 폭발 |
| 2025 | $80~$200 (스탠다드), 콤보 $400~$2,000 | 확립된 인기 모프 |

## C. 색상/기저색 형질 (Color Traits)

크레스티드 게코의 기저색은 **파이어드 업(fired up)** 상태에서 판별한다.

### 4대 기저색 그룹

| 그룹 | 색상 범위 | 설명 |
|------|-----------|------|
| **다크 (Dark)** | 브라운, 그레이, 블랙 | 가장 흔한 기저색 |
| **레드 (Red)** | 오렌지-레드 스펙트럼. 네온, 파이어트럭, 살몬, 핑크 세부 | "성배(holy grail)" — 가장 높은 수요 |
| **옐로우 (Yellow)** | 크림~밝은 카나리아 | 시트러스(Citrus)로도 불림 |
| **라벤더 (Lavender)** | 보라·푸른 톤. 블루 페이즈, 그레이 변형 | 이리도포어 반사 효과. 희귀 |

### 주요 색상 모프

| 모프 | 설명 | 가격대 |
|------|------|--------|
| **레드 (Red)** | 진한 빨간 기저. 익스트림 할리퀸 + 레드가 최고 인기 | 프리미엄 |
| **오렌지 (Orange)** | 순수 오렌지 톤. 진짜 오렌지는 생각보다 드묾 | 중상 |
| **탠저린/하이퍼탠저린 (Tangerine)** | 강렬한 오렌지. 선별교배의 정점 | 프리미엄 |
| **크림시클 (Creamsicle)** | 크림 + 오렌지 조합 | 중 |
| **할로윈 (Halloween)** | **진정한 블랙 + 오렌지**. 브라운/옐로우 대체 불가 | 중상 |
| **라벤더 (Lavender)** | 보라/푸른 톤. 블루 페이즈 포함 | 중상~프리미엄 |
| **올리브 (Olive)** | 올리브/녹색 톤 | 중 |
| **벅스킨 (Buckskin)** | 황갈색 | 엔트리 |
| **사쿠라 (Sakura)** | 분홍빛 | 중상 |

### 파이어드(Fired) vs 언파이어드(Unfired)

| 상태 | 설명 |
|------|------|
| **Fired up** | 가장 어둡고, 선명하고, 대비가 강하고, 채도가 높음. **기저색 판별 기준** |
| **Fired down** | 창백하고, 둔하고, 색이 빠진 상태. 일부 모프에서 완전히 다른 개체처럼 보임 |

## D. 구조적 형질 (Structural Traits)

| 형질 | 설명 |
|------|------|
| **크라운드 (Crowned)** | 머리가 유독 크고 크레스트가 발달. 등급: Poor/Bald → Average → Good → Crowned |
| **프린지 (Fringe)** | 뒷다리 뒤쪽에 크림/오렌지/황색 장식 스케일이 발까지 연장 |
| **니캡스 (Kneecaps)** | 무릎 부위 스케일이 밝게 하이라이트 |
| **포트홀즈 (Portholes)** | 하부 측면에 흰색 점. 유체 시 작고 성장하며 "블룸"(확장) |
| **퍼드/퍼리 (Furred/Furry)** | 등면 돌출 스케일이 2~4줄. 핀스트라이프의 다열 버전 |
| **블러시 (Blush)** | 턱 아래 오렌지/빨간 착색 |
| **소프트 스케일 (Soft Scale)** | 부드러운 피부 질감. 카푸치노/릴리와 별개로 존재 |

## E. 특수 유전자/프로젝트

### C2 유전자

- **정의**: 크림 × 크림 교배에서 발견된 유전적 강화 인자
- **효과**: "더 선명한 색상 + 탁한 갈색 감소" → 깨끗한 색감
- **상태**: 프로젝트 단계, 정식 유전 방식 미확인

### 콜드 퓨전 (Cold Fusion)

- 하이웨이/대립유전자 복합체 관련 프로젝트
- 블루/라벤더 색조의 개체 → $800~$1,500
- 정확한 유전 방식 연구 진행 중
