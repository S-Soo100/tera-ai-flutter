> 관련 그래프 엔티티: `ent-species-crested` (→ `products/app/data/graph.json`)

## 1. 정의와 생물학적 원리

### 아잔틱(Axanthic)이란?

- 어원: **a-(없다) + xanthos(노란색)** = "노란색이 없는"
- 색소세포(chromatophore)의 **잔토포어(xanthophore) 층이 기능하지 않거나 결여**되어 황색·적색 색소가 생산되지 않는 상태
- 크레스티드 게코에서 현재까지 유일하게 증명된 **단순 열성(simple recessive) 돌연변이**

### 색소세포(Chromatophore) 구조

파충류의 피부색은 3층의 색소세포로 결정된다:

| 층 | 이름 | 역할 | 아잔틱에서 |
|----|------|------|-----------|
| 최상층 | **잔토포어 (Xanthophore)** | 황색·적색 색소 생산. 오렌지·빨간 색소 과립은 에리트로포어(Erythrophore)로도 불림 | **결여 또는 기능 상실** |
| 중간층 | **이리도포어 (Iridophore)** | 빛을 반사·굴절 → 파란색·녹색 색조 생성 | 정상 작동 → 은색·청색 색조 가능 |
| 최하층 | **멜라노포어 (Melanophore)** | 멜라닌 생산 → 검은색~갈색 | **정상 작동** → 검정·갈색 유지 |

### "브라운 아웃(Brown-out)" 현상

아잔틱이라 해도 완전한 흑백만 나오지는 않음:
- 멜라노포어가 정상이므로 **갈색 색소**는 여전히 생산됨
- 황색·적색이 사라진 자리에서 멜라닌 계열이 남아 **갈색·적갈색 톤**이 관찰될 수 있음
- 이것이 소위 "브라운 아잔틱" — 순수 흑백이 아닌 갈색 기조를 보이는 아잔틱 개체

## 2. 기원과 역사

### 최초 발견

- **발견지**: 영국 (UK)
- **미국 도입**: 영국에서 발견된 아잔틱 개체를 미국 브리더 **Brian Butler (Altitude Exotics)** 가 구입
- Brian Butler가 아잔틱을 본격적으로 번식·보급하면서 미국 시장에서 주목받기 시작

### 3대 아잔틱 라인

짧은 기간에 3개의 독립적인 아잔틱 라인이 거의 동시에 등장했다. 이는 열성 유전자 특성상 **헤테로 캐리어(het)가 커뮤니티에 이미 퍼져 있었기 때문**으로 추정된다.

| 라인 | 기원/설명 | 특이사항 |
|------|-----------|----------|
| **AE (Altitude Exotics)** | Brian Butler가 UK 라인에서 확립. 가장 잘 알려진 라인 | 밝은 색상의 아잔틱 라인 개발, 아잔틱 릴리 화이트 최초 생산 (2019) |
| **MSL** | Eddie Shelton 관련. 기원 개체의 역사가 **약 15년 전**까지 거슬러 올라감 | 다른 어떤 아잔틱보다 오래된 기록을 가진 파운더 가능성 |
| **Obscurial** | 별도로 등장한 제3의 라인 | AE/MSL과 비교 시 미세한 색상 차이 보고 |

### 라인 간 호환성

- 3개 라인이 거의 동시기에 등장한 것 자체가 **같은 유전자 좌위의 돌연변이**일 가능성을 시사
- 실제 교배 결과, **라인 간 호환되는 것으로 보고됨** — 서로 다른 라인의 het 교배에서 비주얼 아잔틱 생산 확인
- 다만 각 라인마다 약간의 외형 차이(황색·적색 잔류 정도)가 존재

## 3. 유전학

### 유전 방식

- **단순 열성 (Simple Recessive)**
- 비주얼(표현형)이 나타나려면 **양쪽 부모 모두**에게서 아잔틱 대립유전자를 받아야 함
- 크레스티드 게코에서 증명된 유일한 단순 열성 형질

### 유전자형 표기

| 유전자형 | 표현형 | 설명 |
|----------|--------|------|
| **ii** (동형접합 열성) | **비주얼 아잔틱** | 아잔틱 외형 발현. 자손에게 반드시 열성 대립유전자 1개 전달 |
| **Ii** (이형접합) | **100% Het 아잔틱** | 외형은 노멀이지만 아잔틱 유전자를 확실히 보유 |
| **II** (동형접합 우성) | **노멀** | 아잔틱 유전자 없음 |

### 교배 결과표

| 교배 조합 | 예상 결과 |
|-----------|-----------|
| 비주얼 × 비주얼 | **100% 비주얼 아잔틱** |
| 비주얼 × 100% Het | **50% 비주얼**, 50% 100% Het |
| 100% Het × 100% Het | 25% 비주얼, **50% 66% Pos Het**, 25% 노멀 |
| 비주얼 × 노멀 | **100% 100% Het** (외형은 전부 노멀) |
| 100% Het × 노멀 | **50% 50% Pos Het**, 50% 노멀 (외형 구분 불가) |

### "Possible Het" 확률

- **66% Pos Het**: Het × Het에서 태어난 노멀 외형 개체 — 실제 Het일 확률 66%
- **50% Pos Het**: Het × Normal에서 태어난 개체 — Het일 확률 50%
- 외형만으로 het 여부 판별 불가 → 테스트 메이팅(test mating) 또는 계통 추적 필요

## 4. 외형 특징

### 기본 색상

- **주 색상**: 검은색, 흰색, 회색, 은색, 갈색 계열만 발현
- 황색·적색·오렌지가 없어 **모노크롬(단색조)** 외형
- 이리도포어의 빛 반사 효과로 일부 개체에서 **은색·청색 색조** 관찰

### 파이어드(Fired) vs 언파이어드(Unfired)

크레스티드 게코는 환경에 따라 색이 변하는데, 아잔틱에서도 동일하게 나타남:

| 상태 | 아잔틱 외형 |
|------|------------|
| **Fired up** (활성) | 가장 어둡고 대비가 강함. 검은색~짙은 회색 바탕 + 선명한 흰색 패턴 |
| **Fired down** (비활성) | 밝은 회색~은색으로 변함. 패턴 대비 감소, 전반적으로 창백 |

### 라인별 외형 차이

| 라인 | 특징 |
|------|------|
| **AE** | 밝은 색상의 아잔틱 라인 존재. 일부 개체에서 연한 청색 색조 보고 |
| **MSL** | 표준적인 흑백~갈색. 일부 미세한 황색/적색 잔류 |
| **Obscurial** | AE/MSL과 미세한 차이. 구체적 비교 데이터는 아직 제한적 |

### 하이포 아잔틱 (Hypo Axanthic)

- 하이포(Hypo) 형질과 결합 시 멜라닌 감소 → **은색·밝은 색조의 아잔틱** 생산
- "실버(silver)" 외형으로 불리며, 가장 밝은 아잔틱 형태

## 5. 주요 콤보 모프

| 콤보 | 구성 | 특징 |
|------|------|------|
| **아잔틱 릴리 화이트 (Axanthic Lilly White)** | Axanthic + Lilly White | 2019년 Altitude Exotics 최초 생산. 은색-흰색 바탕 + 릴리 화이트 패턴. "극도로 희귀" |
| **아잔틱 카푸치노 (Axanthic Cappuccino)** | Axanthic + Cappuccino | 모노크롬 + 카푸치노의 농도감 → 회색·미묘한 갈색 조합 |
| **아잔틱 프라푸치노 (Axanthic Frappuccino)** | Axanthic + Super Cappuccino | 그레이스케일 + 밝은 톤 → 고대비·몽환적 외형 |
| **아잔틱 팬텀 (Axanthic Phantom)** | Axanthic + Phantom | 팬텀이 흰색 패턴을 억제하면서 전체적으로 매우 어두운 외형. 주로 검은색~갈색 |
| **아잔틱 핀스트라이프 (Axanthic Pinstripe)** | Axanthic + Pinstripe | 등 양옆의 흰색 줄무늬가 더 뚜렷하게 강조됨 |
| **아잔틱 하이포 (Axanthic Hypo)** | Axanthic + Hypo | 멜라닌 감소 → 은빛 밝은 기조의 아잔틱 |
| **콜드 퓨전 (Cold Fusion)** | Axanthic 계열 콤보 | 복합 형질 조합 프로젝트. 구체적 조합은 브리더마다 다름 |

## 6. Het 아잔틱 식별

외형만으로 100% Het 아잔틱을 식별하는 것은 **불가능**에 가깝다:

- Het 개체는 완전한 노멀 외형
- **식별 방법**:
  1. **계통 추적 (Lineage)** — 부모가 비주얼 아잔틱이면 자손은 최소 100% Het
  2. **테스트 메이팅 (Test Mating)** — 비주얼 아잔틱과 교배 시 비주얼 자손이 나오면 Het 확정
  3. **통계적 추정** — 여러 클러치에서 비주얼 비율로 추정 (표본 클수록 정확)
- 유전자 검사(DNA test)는 아직 크레스티드 게코 아잔틱에 대해 상용화되지 않음

## 7. 건강 및 사육

### 건강 이슈

- 아잔틱 자체로 인한 **특별한 건강 문제 보고 없음**
- 색소 결핍이 면역·시력·수명에 영향을 준다는 증거 없음
- 일반 크레스티드 게코와 **동일한 사육 환경**으로 관리 가능

### 사육 조건 (노멀과 동일)

| 항목 | 조건 |
|------|------|
| 온도 | 22~26°C (야간 18~22°C) |
| 습도 | 60~80% |
| 조명 | UVB 선택적 (필수는 아님) |
| 먹이 | CGD(크레스티드 게코 다이어트) + 간헐적 곤충 급여 |
| 수명 | 15~20년 (일반 크레스티드 게코와 동일) |

### 번식 시 유의점

- 열성이므로 비주얼 생산까지 **최소 2세대** 소요 가능
- Het × Het 교배에서도 비주얼 확률 25% → 많은 클러치 필요
- 라인 간 호환 여부를 반드시 확인 후 교배 (대부분 호환되나 결과 모니터링 필요)

## 9. 사업적 시사점

Tera AI 관점에서의 포인트:

- **유일한 증명된 열성 모프**: 크레스티드 게코 유전학의 핵심 — 교육 콘텐츠·지식 그래프 확장 시 "유전학 입문"의 대표 사례로 활용 가능
- **브리딩 프로젝트 도구**: 아잔틱 번식은 계통 추적·확률 계산이 핵심 → AI 앱에서 **브리딩 계획기(breeding planner)** 기능의 킬러 유스케이스
- **건강 이슈 없음**: 세이블 슈퍼폼과 마찬가지로 사육 난이도가 노멀과 동일 → 사육장 환경 세팅에 추가 고려사항 없음
- **콤보 확장성**: 카푸치노·릴리 화이트·팬텀 등과의 조합이 활발 → 모프 조합 시뮬레이션 기능 수요 있음
