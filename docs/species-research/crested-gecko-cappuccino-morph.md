## 1. 기원과 역사

### 발견

- **발견지**: 한국, **Reptile City Korea (RCK)**
- **시기**: 2020년경
- **경위**: RCK가 아잔틱 유사 외형의 개체들을 수 세대에 걸쳐 번식하던 중, 의도치 않게 유전적 다양성이 축적. 팬텀 유전자와 우연히 결합되면서 최초의 카푸치노 팬텀이 만들어짐 — 이때까지 새로운 유전 형질인 줄 몰랐음

### 타임라인

| 시기 | 사건 |
|------|------|
| **2020년** | RCK, 카푸치노 2마리를 교배 → 세계 최초 "멜라니스틱" 크레스티드 게코 확인 |
| **2020년 11월** | 최초 멜라니스틱(슈퍼 카푸치노) 해칭 시도 — 알 안에서 사망 |

| **2021년 중반** | RCK가 다른 브리더에게 카푸치노 판매 시작, 공개 발표 |

| **2024~2025년** | 가격 정상화, 브리더 네트워크 확대 |

### 명칭 변천

1. **멜라니스틱(Melanistic)** — 초기 명칭. 하지만 실제로 멜라닌 과잉 생산이 아니라 피부 구조·색소층 변화이므로 부정확
2. **카푸치노(Cappuccino)** — 이형접합(het) 개체의 공식 명칭
3. **슈퍼 카푸치노(Super Cappuccino)** — 동형접합 개체의 공식 명칭 (= 멜라니스틱)
4. **프라푸치노(Frappuccino)** — 카푸치노 × 릴리 화이트 콤보 명칭 (인스타 브리더 @reptilectrix_exotics가 명명)

## 2. 유전학

### 유전 방식

- **불완전우성 (Incomplete Dominant)**
- 이형접합(het) = 카푸치노 표현형
- 동형접합(homo) = 슈퍼 카푸치노(멜라니스틱) 표현형
- 유전자 표기: **C**

### 대립유전자 복합체

카푸치노는 **세이블(Sable)**, **하이웨이(Highway)**와 같은 유전자 좌위(locus)를 공유한다:
- 카푸치노 × 세이블 → **루왁(Luwak)** (복합 헤테로)
- 각각 별도의 대립유전자이지만 같은 위치에 있음
- 릴리 화이트, 팬텀, 아잔틱과는 **비대립(non-allelic)** → 독립적으로 조합 가능

### 교배 결과표

| 교배 조합 | 예상 결과 |
|-----------|-----------|
| Cappuccino × Normal | 50% Cappuccino, 50% Normal |
| Cappuccino × Cappuccino | 25% **Super Cappuccino**, 50% Cappuccino, 25% Normal |
| Cappuccino × Sable | **Luwak** (복합 헤테로) |
| Cappuccino × Lilly White | **Frappuccino** (두 불완전우성 유전자 동시 발현) |

## 3. 외형 특징

### 이형접합 카푸치노 (Cappuccino)

#### 해칭~유체
- 꼬리 기저부가 **밝고 선명한 흰색**, 꼬리 끝은 매우 어두움 (패턴 거의 없음)
- **Y자 패턴**(꼬리 기저부) — 가장 신뢰할 수 있는 카푸치노 식별 마커 (성장하면서 희미해질 수 있음)
- 척추를 따라 흰색 반점(white flecks)
- 회색·검은색·짙은 갈색 기저

#### 성체
- 유체 시절의 색상이 **흐려지고 섞임**
- 대부분 등면 패턴 소실
- 핀스트라이프가 넓어지며 흰색 → 황색으로 변화 가능
- 일부 개체에서 **거의 검은 혀** 관찰
- 표현 강도에 개체 차이가 큼 (high expression vs low expression)

### 슈퍼 카푸치노 / 멜라니스틱 (Super Cappuccino)

| 항목 | 특징 |
|------|------|
| **피부** | **실크 질감(silky)**, 스케일 구조 감소, 반투명(translucent) — "피부 한 층이 빠진 것 같은" 느낌 |
| **눈** | 해칭 시 **순흑색(solid black)**, 성장하면서 마블링·스네이크 아이 발달 |
| **크레스트** | 최소화됨 |
| **꼬리** | 보통보다 긴 편 |
| **패턴** | 거의 없음. 성장하면서 달마시안 스팟이나 불규칙한 흰색/탄색 영역 발생 가능 |
| **파이어링** | 파이어드 업/다운 가능, 성장에 따라 변화 |

## 4. 건강 이슈 ⚠️

카푸치노 모프에서 **가장 중요하고 논쟁적인 부분**. 건강 문제는 **동형접합(슈퍼 카푸치노)에서만** 발생하며, 이형접합 카푸치노와 콤보 모프는 영향 없음.

### 확인된 구조적 문제

약 100마리의 슈퍼 카푸치노/소락 데이터 기준 (Lil Monsters Reptiles Foundation Genetics PSA):

| 증상 | 발생률 | 설명 |
|------|--------|------|
| **콧구멍 축소 (Reduced Nostrils)** | ~11% | 콧구멍이 작아지거나 완전히 막힘. 모든 라인(한국/미국/영국)에서 발생 |
| **스펙터클 아이 (Spectacle Eye)** | ~3% | 눈과 눈 비늘 사이에 체액 축적. 누관 형성/폐쇄 문제 |

### 중요 사실

- **이형접합 카푸치노에서는 발생하지 않음**
- 근친 교배와 무관 — 아웃크로스 개체에서도 발생
- 저온 배양(69~71°F)이 콧구멍 문제를 줄일 수 있다는 보고
- 유전적 질환이라기보다 **구조적 발달 문제**에 가까움

### 디벙크된 루머

| 루머 | 사실 |
|------|------|
| "슈퍼 카푸치노는 스스로 해칭 못 함" | ❌ RCK가 수십 마리 자연 해칭 기록 |
| "전부 암컷" | ❌ 오히려 수컷이 약간 더 많음 |
| "실험실 제작/하이브리드" | ❌ 명백히 거짓 |
| "전부 불임" | ❌ 확정되지 않음, 장기 번식 시험 진행 중 |

### MorphMarket 판매 금지

- **슈퍼 카푸치노(동형접합)의 판매가 MorphMarket에서 금지**됨
- 이형접합 카푸치노, 프라푸치노 등 콤보 모프는 판매 가능
- 윤리적 논쟁 진행 중 — 의도적 슈퍼 카푸치노 생산을 피하자는 움직임

## 6. 하이웨이(Highway) — 복합체의 세 번째 대립유전자

### 개요

- **다른 이름**: Cold Fusion
- 세이블·카푸치노와 **같은 유전자 좌위의 또 다른 대립유전자**
- 불완전우성으로 추정
- 카푸치노·세이블에 비해 정보가 제한적 — 초기 프로젝트 단계

### 슈퍼 하이웨이 건강 이슈

- 슈퍼 하이웨이에서 **콧구멍 축소·건강 저하**가 슈퍼 카푸치노와 유사하게 보고됨
- 이것이 하이웨이가 카푸치노/세이블과 같은 대립유전자 복합체임을 뒷받침하는 증거 중 하나

### 대립유전자 복합체 전체 정리

| 대립유전자 | 발견지 | 슈퍼폼 건강 |
|-----------|--------|------------|
| **카푸치노** | 한국 (RCK) | ⚠️ 콧구멍 축소 ~11%, 스펙터클 아이 ~3% |
| **세이블** | 미국 (Gecko Haven) | ✅ 문제 보고 없음 |
| **하이웨이** | 미국 | ⚠️ 슈퍼 카푸치노와 유사한 문제 보고 |

→ 세이블이 슈퍼폼 건강 면에서 가장 안전한 대립유전자

## 9. 사육 참고

### 이형접합 카푸치노

일반 크레스티드 게코와 **동일한 사육 환경**:

| 항목 | 조건 |
|------|------|
| 온도 | 22~26°C (야간 18~22°C) |
| 습도 | 50~80% 사이클링 |
| 사육장 | 수직 공간 + 등반 구조물 + 은신처 |
| 먹이 | CGD + 간헐적 곤충 급여 |

### 슈퍼 카푸치노 특별 주의사항

- **수분 공급 철저** — 피부 건강 유지 핵심
- **부드러운 핸들링** — 피부가 연약함(실크 질감)
- **먼지 없는 환경** — 축소된 콧구멍의 호흡기 부담 방지
- **탈피 모니터링** — 피부 구조 때문에 탈피 장애 가능
- **눈 상태 관찰** — 스펙터클 아이 조기 발견 시 수의사 상담
- 특별한 식이 요구는 없음

## 10. 사업적 시사점

- **한국 발견 모프**: RCK가 최초 발견 → Tera AI 앱에서 "한국에서 시작된 유전 모프" 스토리텔링 가능
- **윤리적 이슈의 교육 기회**: 슈퍼폼 건강 문제 → 앱에서 "책임 있는 번식" 교육 콘텐츠로 활용
- **브리딩 플래너 핵심 사례**: 카푸치노 × 세이블 = 루왁 (건강한 대안) vs 카푸치노 × 카푸치노 = 슈퍼 카푸치노 (건강 리스크) → 교배 조합 추천 엔진의 킬러 시나리오
- **가격 폭락 추세**: 초기 $25,000+ → 현재 $300~$1,000 수준. 한국 시장은 더 저렴 → 대중화된 프리미엄 모프
