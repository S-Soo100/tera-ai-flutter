### 영원회귀 지점 (반복 고통)

1. **홈 → 사육 가이드 카드 → 위키 → 종 칩 다시 선택** — `context.go('/wiki')`만 호출, 종 정보 미전달. 매번 칩 재선택
2. **검색 → "상세 정보" → 위키 → 종 재선택** — 검색 맥락 완전 유실
3. **개체 상세 → "위키 보기" → 위키 → 종 재선택** — `context.push('/wiki')`만 호출, 해당 종 미선택
4. **모프 계산기 접근 최소 3탭** — 위키 → 종 선택 → 그리드에서 찾기. 홈/내개체에서 바로가기 없음
